# Zerodha Clone

A full-stack Zerodha trading platform clone built using the MERN stack.

## Live Demo

Frontend: https://zerodha-frontend-ozuo.onrender.com/

Backend API: https://zerodha-clone-geps.onrender.com/

---

## Features

* User Authentication (Login & Signup)
* Holdings Management
* Positions Tracking
* Order Management
* Dashboard Interface
* MongoDB Database Integration
* REST API Backend
* Responsive User Interface

---

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### Deployment

* Render
* MongoDB Atlas
* GitHub

## API Endpoints

### Holdings


### Positions

```http

```

### Orders

```http
POST /newOrder
```

### Authentication

```http
POST /auth/signup
```
